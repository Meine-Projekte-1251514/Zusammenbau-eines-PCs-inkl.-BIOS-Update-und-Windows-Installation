# Zusammenbau eines PCs inkl. BIOS-Update und Windows-Installation

Zusammenbau eines PCs inkl. BIOS-Update und Windows-Installation

# Projektübersicht

  Es wurde ein PC eigenständig zusammengestellt, aufgebaut und in Betrieb genommen. Neben Hardwaremontage umfasst das Projekt die Fehleranalyse bei Startproblemen sowie die Aktualisierung der BIOS-Firmware.

# Ziel & Motivation

  Verständnis für Hardwarekompatibilität

  Praktische Erfahrung im PC-Zusammenbau 

  Eigenständige Fehlerdiagnose

  Installation und Konfiguration von Windows Betriebssystemen

  # Systemumgebung

  Selbst zusammengestellte Hardware (CPU, Mainboard, RAM, GPU, PSU)

  Betriebssystem: Windows 10 

  # Probleme & Fehleranalyse

  System scheiterte den POST

  Analyse möglichen Ursachen (Error LED)

  Feststellung einer veralteten BIOS-Version

  # Lösung

  Durchführen eines BIOS-Firmware Updates (Flashen)

  Erneuter Systemstart nach dem Update

  Erfolgreicher Abschluss des POST 

  Fortsetzung der Windows installation 

  #Ergebnis 

  Voll funktionsfähiges PC System 

  Stabiler Betrieb nach BIOS Update

  Erfolgreich installiertes und konfiguriertes Windows

  #Erkenntnisse & Lernerfolge 

  Hardwaremontage und -prüfung 

  Bedeutung von BIOS firmware für Systemstabilität

  Systematisches Vorgehen bei Bootproblemen 

  Umgang mit kritischen Systemupdates
